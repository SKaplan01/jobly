# react_jobly

To see a live demo: https://jobilly.herokuapp.com/companies

**To clone this repository:**

*Set up the backend:*
1. cd "backend"
2. createdb 'jobly'
3. run seed.js
4. `npm install`
5. `nodemon`

*Set up the frontend:*
1. cd "frontend"
2. npm install
3. npm start

All routes are prefixed with `/api` so to fetch posts the route is `GET /api/posts`
